# 部署
## windows上
注：make setup报错；
全新部署（选择1）：
```
scripts\docker-setup.bat
```
```
   Select Deployment Option
========================================

  1. Full Deployment (Setup + Build + Start)
  2. Start Services (Production)
  3. Start Services (Development)
  4. Stop All Services
  5. Restart Services
  6. View Service Status
  7. View Logs
  8. Clean All Data
  0. Exit
```


注意：有代理包安装报错的，临时关闭代理；
```
set http_proxy=
set https_proxy=
```

缺点：
```
会构建镜像（非常大，并且耗时，约40min）：image: tianshu-backend:latest(15.23G左右)
```
